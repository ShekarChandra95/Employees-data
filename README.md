# Employee Attendance Analysis using Python

## 📌 Overview
This project analyzes employee attendance data to identify absentee trends and improve workforce management.

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib

## 📊 Analysis Performed
- Attendance status distribution
- Department-wise absenteeism
- Trend analysis

## 📈 Key Insights
- Identified departments with high absentee rates
- Highlighted attendance patterns

## 🚀 How to Run
pip install -r requirements.txt  
python src/analysis.py
